## MODIFIED Requirements

### Requirement: Project switching from sidebar
The project switching capability in the sidebar SHALL now be positioned at the top of the sidebar for improved discoverability and accessibility, replacing the mid-sidebar location.

#### Scenario: Project switcher is top-positioned
- **WHEN** user views the sidebar
- **THEN** the project switcher appears below the header (at the top of navigation content)

#### Scenario: Active project is always visible
- **WHEN** user is in the application
- **THEN** the current active project is displayed prominently in the top project switcher section

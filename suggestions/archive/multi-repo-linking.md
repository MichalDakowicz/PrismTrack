# Multi-Repo Linking

## Overview
Enhance project-repository linking to support complex multi-repository workflows.

## Motivation
Currently projects can link to repositories, but the experience may be basic. This would enable more sophisticated workflows.

## Potential Features

### Enhanced Linking
- Search and link multiple repositories to a project
- Repository priority ordering
- Filter issues/PRs by repository

### Cross-Repository Views
- Aggregate views across all linked repos
- Unified search
- Cross-repo dependency tracking

### Repository Groups
- Group repositories by team/org
- Quick toggle visibility
- Bulk operations

## Priority
HIGH - Core multi-repo functionality

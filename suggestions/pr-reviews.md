# Full PR Reviews

## Overview
Enhance the pull request view to include full review functionality, not just listing PRs.

## Motivation
Currently PRs are displayed as cards with basic info. A full review experience would provide more utility.

## Potential Features

### PR Detail View
- Full PR description with markdown rendering
- File changes diff viewer
- Review comments thread
- CI/CD status checks

### Review Actions
- Approve / Request changes / Comment
- Inline code suggestions
- Review status summary

### Activity Timeline
- Review events
- Comments
- Status changes
- Check runs

### Conversation View
- Unified PR conversation
- Threaded comments
- Resolve/reopen threads

## Priority
HIGH - Core GitHub workflow feature
